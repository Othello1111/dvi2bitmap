dvi2bitmap font samples

The following are examples of the output of dvi2bitmap, with different arguments and modes.

  • modetest: this is the modes test file, extracted from the Metafont modes.mf file, by Matt Swift.
  • modetest-maths: this is the maths part of modetest by itself
  • modetest-maths-concmath: this is the same as modetest-maths, but including \usepackage{concmath}, so that the Concrete Maths font is used.
